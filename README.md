## How to Run this Program Locally 🖥️
1. Download all py and env files into one folder
   ```cmd
   Blog_Generator.py
   ```
2. Install openai
   ```cmd
   pip install openai
   ```
3. Must obtain an API key from OpenAI, that you can put in the env file
   ```cmd
   Key.env
   ```
4. Run Program
